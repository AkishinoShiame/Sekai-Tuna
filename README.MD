# 世界はこれから繋がっている

このプロジェクトは「世界はこれから繋がっている」と言うタイトルの夢。　このプロジェクトは二つ目標であり。
一つ、物語を作る。
一つ、恋愛アドベンチャーゲームを作る。
大体このイメージ通りのプロジェクト。

始めたの時間はこれから、
終わり時間は今わからない。

さあ、これからガンガン行きましょう！
